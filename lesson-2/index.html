ОСНОВНЫЕ ТЕГИ HTML

1) СТРУКТУРА Web-страницы
<html></html> Начало и конец страницы
<head></head> Описание страницы, содержащее различную информацию не отображаемую в теле документа. Здесь распологается тег имени страницы и теги для поисковых машин
<title></title> Имя страницы
<body></body> Определяет видимую часть страницы
2) АТРИБУТЫ ТЕЛА ДОКУМЕНТА
Значения атрибутов
background="URL"> Фоновое изображение
bgcolor="#RRGGBB"> Цвет фона документа, используя значения цвета в виде RRGGBB - пример: FF0000 - красный цвет. red,blue,#FF0000 и др.
text="#RRGGBB"> Цвет текста документа, используя значения цвета в виде RRGGBB - пример: 000000 - черный цвет.
link="#RRGGBB"> Устанавливает цвет гиперссылок, используя значение цвета в виде RRGGBB - пример: 00FF00 - зеленый цвет.
vlink="#RRGGBB"> Устанавливает цвет гиперссылок на которых вы уже побвали, используя значения в виде RRGGBB - пример: 666666 - серый цвет.
alink="#RRGGBB"> Устанавливает цвет гиперссылок при нажатии.
3)ТЕГИ ДЛЯ ФОРМАТИРОВАНИЯ ТЕКСТА
Значения атрибутов
<h?></h?> Создает заголовок (уровни от 1 до 6)
<h?align="*"></h> Заголовок с выравниванием left,right, center
<p></p> Создает новый параграф
<p align="*"></p> Выравнивает параграф относительно одной из сторон документа left,right, center,justify
<br> Вставляет перевод строки
<center></center> Выравнивание по центру
<div align="*"></p> Форматирование больших блоков текста страницы, также используется в таблицах стилей left,right, center,justify
4) ТЕГИ ДЛЯ ФОРМИРОВАНИЯ ШРИФТА
Значения атрибутов
<b<</b> Жирный
<i></i> Курсив
<u></u> Подчеркнутый
<sup></sup> Верхний индекс - пример а2
<sub></sub> Нижний индекс - пример: a2
<fond size=?></font> Размер шрифта в пределах от 1 до 7.
